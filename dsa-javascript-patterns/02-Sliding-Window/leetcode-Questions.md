⚡ Smart Practice Strategy

1. **Identify the Type**: Is the window size fixed or does it change based on a condition?
2. **Expansion**: Move the `right` pointer to expand the window and include more elements.
3. **Contraction**: Move the `left` pointer to shrink the window once the condition is met (or violated).
4. **State Management**: Use a Hash Map, Frequency Array, or a Simple Variable to track the current window's "state" (sum, count, unique chars).
